# Project Set Up
1. Download zip file
2. Extract it in XAMPP-> htdocs folder
3. open file in VSCode
4. open XAMPP server
5. Create Database Name is " hospital "
6. import Database file from "DATABASE" folder (hospital.sql) in database(hospital)
7. Now run on localhost
